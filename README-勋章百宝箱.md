## 基于GM论坛开发的勋章管理软件

拥有以下功能  
1.默认关闭回收功能  
2.徽章按类型排序  
3.保存/还原徽章顺序  
4.勋章一键续期  
5.勋章收益统计功能  
6.所有勋章一键续  
7.一键关闭赠礼和咒术类勋章显示
